The constructor of Attribute, the base class for any concrete Attribute.

This constructor will ensure that the memory reserved through AttributeSettings * used in construction will be freed (allows the use of new in the constructor).

Parameters
baseNodeThe pointer to the Node to which this copy of Attribute is assigned.
baseTreeThe pointer to the ImageTree to which the Node baseNode belongs to.
settingsThe pointer to the AttributeSettings holding the necessary settings for this Attribute. A copy of these settings is assigned to the Attribute.
deleteSettings(optional) Ensures that the memory of settings is freed after the call to this constructor. If true, the settings will be deleted, use false if settings need to be stored for later processing.
