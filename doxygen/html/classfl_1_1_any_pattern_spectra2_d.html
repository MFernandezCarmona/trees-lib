The default constructor for AnyPatternSpectra2D.

Initializes AnyPatternSpectra2D and associates all the related objects for pattern spectra calculation

Parameters
baseNodeThe Node in the ImageTree for which the pattern spectrum is to be calculated. Has to belong to baseTree
baseTreeThe ImageTree which is used for the calculation of the pattern spectrum
settingsThe PatternSpectra2DSettings containing the settings which are to be applied during pattern spectrum calculation.
deleteSettings(optional) Ensures that the memory of settings is freed after the call to this constructor. If true, the settings will be deleted, use false if settings need to be stored for later processing.
Note
Pattern spectra are not to be created directly, but rather associated to the whole tree through ImageTree::addPatternSpectra2DToTree which will then call the appropriate pattern spectra constructors
In order to handle the area normalization, a call to this constructor will associate an AreaAttribute to the whole ImageTree if called correctly.
fl::AnyPatternSpectra2D::~AnyPatternSpectra2D ( void  )
pure virtual

The destructor for AnyPatternSpectra2D.

Ensures the proper destruction of AnyPatternSpectra2D.

Note
The pattern spectra are not to be deleted directly and this destructor should not be called directly. In order to remove pattern spectra from a tree, the function ImageTree::deletePatternSpectra2DFromTree should be used.
If the pattern spectra are deleted through the appropriate functions, this destructor will also delete any additional AreaAttributes from the ImageTree.

virtual

Gets the reference to the filled out matrix (histogram) holding the PatternSpectra2D values.

Obtain the filled out histogram that is the pattern spectrum. Internally, the function calculatePatternSpectra2D is called.

Parameters
pathDownIf the pattern spectrum is calculated from the parent (ancestor), allows passing an ancestral path down to the original node. Only the values of the Nodes on the path down are calculated into the pattern spectrum.
Note
pathDown would end with the Node for which the descriptor is being calculated. All the child Nodes from this point down are calculated into the pattern spectrum.
Returns
A reference to the filled out histogram holding the pattern spectrum.
