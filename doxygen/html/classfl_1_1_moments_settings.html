Constructor for the MomentsSettings.

Settings for the MomentsAttribute, which will be used in MomentsHolder::setDefaultCastValue() to determine the output value of the MomentsAttribute.

order The maximal order to be stored within MomentsHolder. dcv MomentType defining the type of the moment (default cast value). dp The p order of the moment. This can later be changed, but only so that p + q <= order.

Remarks
Ignored in case of MomentType::kurtosis, MomentType::roundness, MomentType::eccentricity, MomentType::triangularity. dq The q order of the moment. This can later be changed, but only so that p + q <= order.
Ignored in all cases as p as well as MomentType::hu.
