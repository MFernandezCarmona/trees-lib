Detailed Description

template<typename TI>
class fl::distanceL1Integral< TI >

L1 distance functor (integral values).

L1 distance functor operating on multispectral images holding integral values.

Note
It is expected that the differences between consecutive values will fall into the integer range.
Template Parameters
TIInput type of the functor. Type of each single component of the multispectral image. Has to be of integral type (short, int, etc.).
