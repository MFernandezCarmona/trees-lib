Detailed Description

Value of this Attribute is the approximation of the diameter of bounding sphere of the pixel values for the ImageTree multivariate data (multiple images set). From: S. Lefevre, L. Chapel, F. Merciol: "Hyperspectral image classification from multiscale description with constrained connectivity and machine learning" (2014)

Note
This is an advanced case of RangeAttribute approximation for multivariate data.
The calculation of this attribute requires reading the pixel values of a multichannel image. The corresponding images (representing a single multichannel image) has to be assigned with ImageTree::setImages().

This is an increasing attribute, meaning that the value of a Node will always be higher or equal than that of it's children Nodes.

void fl::BoundingSphereDiameterApprox::calculateAttribute ( )
virtual

Triggers the calculation of the BoundingSphereDiameterApprox value for a Node.

Calculation of the RangeAttribute. The calculation is done efficiently, by re-using the values of the child Nodes. Any image element (pixel) position in the ImageTree is examined at most once.

Note
A multivariate image needs to be associated to the ImageTree before this function is called. This can be done by ImageTree::setImages (one image for each dimension of the multivariate image). After the calculation the image can be discarded by ImageTree::unsetImages.
